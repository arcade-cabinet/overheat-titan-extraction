# OVERHEAT: Titan Extraction

Browser-first extraction / resource-management sandbox built with React, Vite, React Three Fiber, and Rapier.

## Start here

If you are an agent or maintainer picking up this repository, use this reading order:

1. [`AGENTS.md`](./AGENTS.md) — master instructions and architecture contract
2. [`docs/README.md`](./docs/README.md) — documentation index by domain
3. [`docs/HANDOFF.md`](./docs/HANDOFF.md) — current implementation state and next priorities
4. [`docs/STANDARDS.md`](./docs/STANDARDS.md) — code quality, brand, and design rules
5. [`docs/AGENTS.md`](./docs/AGENTS.md) — documentation-specific authoring rules

pnpm install
pnpm run dev
pnpm run build
